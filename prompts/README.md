# Prompt Library — Thư Viện Prompt Theo Chủ Đề

Tất cả prompt mẫu được tổ chức theo loại công việc, dựa trên framework 4C.

> **Cách dùng:** Mở prompt phù hợp → điền thông tin vào `[bracket]` → gửi cho AI.

---

## Danh Sách Thư Viện

| Thư mục | Loại công việc | Dùng khi |
|---|---|---|
| [content/](./content/) | Tạo content marketing | Viết caption, blog, script, email sequence |
| [creative/](./creative/) | Sáng tạo & ý tưởng | Brainstorm chiến dịch, concept, tên sản phẩm |
| [operations/](./operations/) | Vận hành & quy trình | Viết SOP, FAQ, báo cáo, checklist |
| [people/](./people/) | Nhân sự & đội nhóm | Tuyển dụng, onboarding, đánh giá nhân viên |
| [relations/](./relations/) | Quan hệ & giao tiếp | Chuẩn bị gặp khách, xử lý phàn nàn, họp team |
| [strategy/](./strategy/) | Chiến lược & phân tích | SWOT, phân tích đối thủ, lập kế hoạch |

---

## Nguyên Tắc Dùng Prompt Library

1. **Điền vào `[bracket]`** — Đừng dùng nguyên mẫu, phải cá nhân hóa
2. **Thêm Context** — Thêm thông tin công ty/sản phẩm/khách hàng của bạn
3. **Tự viết phần cảm xúc** — AI viết nháp, bạn thêm giọng văn và cảm xúc thật
4. **Chấm theo Criteria** — Dùng Criteria trong prompt để tự đánh giá output
5. **Lưu prompt hiệu quả** — Khi tìm được prompt tốt, lưu lại vào thư mục của bạn
