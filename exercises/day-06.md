# Day 06 — Bài Tập: Tư Duy Vòng Lặp

## Mục tiêu

Xác định vòng lặp hiện tại của bạn với AI, thiết lập feedback loop rõ ràng, và lên kế hoạch 30 ngày.

---

## Bài 1: Chẩn đoán vòng lặp hiện tại

Nhìn lại cách bạn dùng AI trong 2 tuần qua. Mỗi task lặp lại, bạn đang ở vòng lặp nào?

| Task lặp lại | Vòng lặp hiện tại | Lý do |
|---|---|---|
| Ví dụ: Viết caption | Đứng yên — dùng cùng prompt, không đo hiệu quả | Không có tiêu chí để so sánh |
| 1. | | |
| 2. | | |
| 3. | | |

**Nhận xét:** Task nào đang ở vòng lặp đứng yên hoặc tụt lùi cần ưu tiên cải thiện nhất?

> (Ghi nhận xét)

## Bài 2: Thiết lập feedback loop cho 1 task

Chọn 1 task đang ở vòng lặp đứng yên. Thiết lập feedback loop 4 bước:

**Task được chọn:** ___

**Bước 1 — Viết Criteria trước:**
```
Output tốt phải:
1.
2.
3.

Output cần viết lại khi:
1.
2.
```

**Bước 2 — Sau khi AI làm xong, chấm theo Criteria:**
```
Điểm: __ / 10
Điểm yếu nhất:
```

**Bước 3 — Điều chỉnh prompt:**
```
Prompt cũ: [tóm tắt]
Thay đổi: [thêm/bỏ/sửa gì]
Prompt mới: [tóm tắt]
```

**Bước 4 — Kết quả sau khi lặp lại:**
```
Điểm lần 2: __ / 10
Cải thiện: có / không / một phần
```
